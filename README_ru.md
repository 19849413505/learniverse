# 🌌 Learniverse (Promax)

**Learniverse** стремится создать **совершенную адаптивную геймифицированную образовательную платформу с открытым исходным кодом нового поколения**. Это не просто инструмент для запоминания слов, а образовательная инфраструктура, объединяющая **механизм графов знаний, автоматический анализ с помощью больших языковых моделей (LLM), алгоритм интервального повторения FSRS с открытым исходным кодом и захватывающий геймифицированный опыт в стиле Duolingo**.

Наше видение — стать **открытой Promax-версией Math Academy**: предоставить каждому возможность превратить любую книгу или документ в детализированную сеть знаний. Благодаря сократовскому эвристическому обучению с использованием ИИ и научному интервальному повторению мы помогаем учащимся достичь истинного «Мастерства» (Mastery).

---

## 🎯 Видение и Преимущества (The Vision & Edge)

### 1. Автоматическое извлечение знаний (Опыт NotebookLM)
Пользователям достаточно загрузить книги (PDF/EPUB) или вставить длинные тексты, и модуль **Кузница Знаний (Knowledge Forge)** автоматически:
- Извлекает ключевые концепции (узлы) и зависимости (ребра) из документа.
- Генерирует **визуальные 2D/3D графы знаний**, позволяя вам окинуть взглядом «лес знаний» перед началом обучения.
- Автоматически создает флэш-карточки (Flashcards) для повторения каждой концепции.

### 2. Сверхэффективный механизм памяти (Алгоритм FSRS)
Попрощайтесь с интуитивным повторением. Мы внедряем самый передовой в мире алгоритм интервального повторения с открытым исходным кодом **FSRS (Free Spaced Repetition Scheduler)**.
- Использует библиотеку `ts-fsrs` для динамического построения вашей **индивидуальной кривой забывания** на основе каждого клика «Забыл/Трудно/Хорошо/Легко».
- Точно рассчитывает следующую дату повторения (Due Date) и стабильность памяти (Stability), заставляя вас повторять материал только на грани забывания, что максимально экономит время.

### 3. Погружение и геймификация в стиле Duolingo
Обучение само по себе противоречит человеческой природе, поэтому мы используем игровые механики для компенсации:
- **Дерево Навыков (Skill Tree)**: Связывает знания вместе, подобно разблокировке дерева навыков в игре.
- **Серии и Опыт (Streak & XP)**: Накапливайте XP за каждое повторение и поддерживайте горячую серию (Streak) за счет регулярных занятий.
- **Лиги (Leagues)**: Вводит механику бронзовых/серебряных/золотых лиг для соревнования с друзьями или учениками со всего мира.
- **Совместные задания (Co-op Quests)**: Объединяйтесь в команды для достижения целей, разрушая изоляцию в обучении.

### 4. На пути к Promax: Сократовский ИИ (Архимед)
Помимо традиционного пассивного режима флэш-карточек «посмотрел вопрос -> проверил ответ», мы внедряем **сократовское эвристическое обучение с помощью ИИ**. Когда вы сталкиваетесь со сложной карточкой, ИИ не просто дает вам ответ. Вместо этого, как терпеливый наставник, он направляет вас к самостоятельному выводу с помощью пошаговых вопросов, достигая истинного глубокого понимания.

## ✨ Текущие возможности (Current Features)

Основываясь на передовой архитектуре, мы завершили создание базовой MVP-версии фронтенда и частичную интеграцию бэкенда, реализовав следующие функции:

- **Адаптивное извлечение знаний**: Поддержка анализа и импорта документов, автоматическая генерация визуальных 2D/3D графов знаний с помощью ИИ.
- **Алгоритм с открытым исходным кодом**: Полная интеграция `ts-fsrs` (Free Spaced Repetition Scheduler) на всех платформах для точного расчета времени повторения.
- **Геймифицированная панель управления**: Дерево навыков в стиле Duolingo, отслеживание прогресса обучения на базе Recharts, визуализация кривой забывания, а также системы серий (Streak & XP) и достижений.
- **Кроссплатформенная поддержка**: Поддержка статического экспорта веб-версии при наличии независимых нативных десктопных приложений Windows/Mac на базе Electron.
- **Управление состоянием и кэшем**: Постоянное хранение данных Zustand и оптимизированный рендеринг дерева знаний React Flow.

## 🔮 Планы на будущее (Future Roadmap)

Мы стремимся постоянно расширять границы платформы, делая ИИ более «понимающим». Наши основные направления развития в будущем:

- **Механизмы глубокого мастерства**: Внедрение предварительных диагностических тестов и проверок зависимостей в графе знаний для реализации автоматизированной «перемежающейся практики» (interleaved practice) и «целенаправленного исправления» (targeted remediation).
- **Эмоциональные наставники с разными характерами (Система Сократ-7)**: Внедрение виртуальных наставников с разными характерами (например, Март 7, Кэ Цин и т. д.) для восстановления погружения посредством диалогового взаимодействия с эмоциональными связями и имитации социального общения после занятий.
- **Персонализированная скорость обучения**: Автоматическое моделирование «скорости обучения» каждого ученика по каждой теме на основе анализа данных для создания по-настоящему адаптивных, индивидуализированных интервалов повторения.
- **Расширение социальных лиг**: Добавление динамических виртуальных учебных групп и механизмов ранжирования в лигах, а также штрафов за качество XP и систем достижений, связанных с эмоциями.

## 💻 Локальная разработка и запуск

Проект управляется с использованием структуры Monorepo.

```bash
# 1. Клонирование репозитория
git clone https://github.com/19849413505/learniverse.git
cd learniverse

# 2. Установка зависимостей
npm install

# 3. Запуск сервера разработки фронтенда
cd packages/frontend
npm run dev &
```

Теперь откройте браузер и перейдите по адресу `http://localhost:3000`, чтобы опробовать платформу!

## 🖥️ Сборка автономного настольного приложения

Вы можете использовать оболочку **Electron**, чтобы в один клик упаковать это веб-приложение в автономную нативную настольную программу (Windows, macOS, Linux) для скачивания пользователями.

👉 **[Подробное руководство по упаковке настольного приложения Learniverse (на английском)](docs/desktop-packaging.md)**

1. **Запустите скрипт автоматической сборки из корневого каталога:** `npm run build:desktop`.
2. Эта команда автоматически соберет весь монорепозиторий и запустит `electron-builder` для генерации настольного приложения в каталоге `packages/desktop/dist/`.
