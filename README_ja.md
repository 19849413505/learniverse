# 🌌 Learniverse (Promax)

**Learniverse**は、**次世代の適応型、ゲーム化された究極のオープンソース学習プラットフォーム**の構築を目指しています。これは単なる単語暗記ツールではなく、**ナレッジグラフエンジン、LLM自動解析、オープンソースのFSRS間隔反復アルゴリズム、そしてDuolingoのような没入型のゲーム化体験**を統合した教育インフラストラクチャです。

私たちのビジョンは、**オープンソース版のMath Academy Promax**になることです。すべての人が、あらゆる書籍やドキュメントを細かい粒度の知識ネットワークに変換する能力を持ち、AIによるソクラテス式ヒューリスティック教育と科学的な間隔反復を通じて、真の「習熟（Mastery）」の境地に到達できるよう支援します。

---

## 🎯 ビジョンと強み (The Vision & Edge)

### 1. 自動化された知識抽出（NotebookLM体験）
ユーザーが書籍（PDF/EPUB）をアップロードするか、長文を貼り付けるだけで、**知識の工房 (Knowledge Forge)** モジュールが自動的に以下の処理を実行します。
- ドキュメントから中心的な概念（ノード）と依存関係（エッジ）を抽出します。
- **視覚化された2D/3Dナレッジグラフ**を生成し、学習前に「知識の森」を俯瞰できるようにします。
- 各概念について、復習用のフラッシュカードを自動生成します。

### 2. 究極に効率的な記憶エンジン（FSRSアルゴリズム）
直感に頼る復習とはお別れです。世界で最も高度なオープンソースの間隔反復アルゴリズムである**FSRS (Free Spaced Repetition Scheduler)**を導入しています。
- `ts-fsrs`ライブラリを採用し、「忘れた/難しい/普通/簡単」をクリックするたびに、あなた専用の**パーソナライズされた忘却曲線**を動的に構築します。
- 次回の復習日（Due Date）と記憶の安定性（Stability）を正確に計算し、忘却の瀬戸際でのみ復習させることで、時間を最大限に節約します。

### 3. Duolingoスタイルの没入感とゲーム化
学習自体は人間の本性に反するため、ゲームのメカニズムを使用してそれを相殺します。
- **学習パス (Skill Tree)**: ゲームのスキルツリーを解放するように知識ポイントを繋ぎます。
- **連続記録と経験値 (Streak & XP)**: 復習するたびにXPを蓄積し、継続的なチェックインによって熱い連続記録を維持します。
- **リーグ (Leagues)**: ブロンズ/シルバー/ゴールドのリーグメカニズムを導入し、友人や世界中の学習者と競い合います。
- **協力クエスト (Co-op Quests)**: チームを組んで目標を達成し、孤立した学習を打ち破ります。

### 4. Promaxへ：ソクラテス式AI（アルキメデス）
従来の「問題を見る -> 答えを確認する」という受動的なフラッシュカードモードに加えて、**ソクラテス式のAIヒューリスティック教育**を統合します。難しいカードに遭遇したとき、AIは単に答えを与えるだけではありません。忍耐強いメンターのように、段階的な質問を通じてあなた自身が答えを導き出せるよう誘導し、真の深い理解を実現します。

## ✨ 現在の機能 (Current Features)

最先端のアーキテクチャに基づいて、現在コアフロントエンドのMVPおよび部分的なバックエンドの統合を完了し、以下の機能を実現しています。

- **適応型知識抽出**: ドキュメントの解析とインポートをサポートし、AIを通じて視覚的な2D/3Dナレッジグラフを自動生成します。
- **オープンソースアルゴリズムエンジン**: すべてのプラットフォームでオープンソースの `ts-fsrs` (Free Spaced Repetition Scheduler) を導入し、復習時間を正確に計算します。
- **ゲーム化されたダッシュボード**: Duolingoスタイルのスキルツリー、Rechartsに基づく学習進捗追跡、忘却曲線の視覚化、およびストリーク（Streak & XP）と実績システム。
- **クロスプラットフォーム対応**: Webの静的エクスポートをサポートしつつ、Electronに基づく独立したWindows/Macネイティブデスクトップアプリを提供します。
- **状態とキャッシュの管理**: Zustandによるデータの永続化と、最適化されたReact Flowナレッジツリーのレンダリング。

## 🔮 今後のロードマップ (Future Roadmap)

私たちは、AIがあなたをより「理解」できるように、プラットフォームの境界を継続的に拡大することに尽力しています。今後の主要な進化ルートは以下の通りです。

- **深い習熟メカニズム**: 事前診断テストとナレッジグラフの依存関係チェックを導入し、自動化された「インターリーブ学習（交互学習）」と「ターゲットを絞った補習」を実現します。
- **マルチペルソナ感情チューター（Socrates-7システム）**: 様々な性格を持つ仮想チューター（三月なのか、刻晴など）を導入し、感情的なつながりや授業後の模擬ソーシャル体験を伴う対話型インタラクションを通じて、没入感を再構築します。
- **パーソナライズされた学習速度**: データフィードバックに基づいて、各トピックにおける各学生の「学習速度」を自動的にモデル化し、真に適応的で個別化された復習間隔を実現します。
- **ソーシャルリーグの拡張**: 動的な仮想学習グループとリーグランキングメカニズムを追加し、XP品質のペナルティと感情にリンクされた実績システムを導入します。

## 💻 ローカルでの開発と起動

このプロジェクトはMonorepo構造を使用して管理されています。

```bash
# 1. リポジトリをクローンする
git clone https://github.com/19849413505/learniverse.git
cd learniverse

# 2. すべての依存関係をインストールする
npm install

# 3. フロントエンド開発サーバーを起動する
cd packages/frontend
npm run dev &
```

ブラウザを開き、`http://localhost:3000` にアクセスして体験してください！

## 🖥️ スタンドアロンのデスクトップアプリのビルド

**Electron**のパッケージングにより、このWebアプリケーションをブラウザ不要のネイティブなクロスプラットフォーム (Windows, macOS, Linux) デスクトッププログラムとしてユーザーに提供することができます。

👉 **[Learniverse デスクトップパッケージングガイドの詳細を見る (英語)](docs/desktop-packaging.md)**
1. **ルートディレクトリから自動ビルドスクリプトを実行します:** `npm run build:desktop`。
2. このコマンドは自動的にプロジェクト全体をビルドし、`electron-builder` を実行して `packages/desktop/dist/` ディレクトリにデスクトップアプリを生成します。
